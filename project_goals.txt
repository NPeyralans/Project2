Programming Project for Active911 Software Developer Position

Goals:
-Make a webpage that has two endpoints: 
	- /query_users
	- /add_user
	- /query_devices: Accepts GET requests with filters
	- /add_device: Accepts POST requests that will add a new device to the database

Devices:
	-Device ID (Primary key automatically produced)
	-Type: "apple" "android" "other"
	-User: maps to users table with foreign key
	-Device name: Options user input

Users:
	-User ID (primary key autogenerated)
	-Email: optional
	-username: unique and not null
	-password_hash

Database
	TABLE: users
	TABLE: devices

Package it in docker and deploy on github
